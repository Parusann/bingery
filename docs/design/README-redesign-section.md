## Redesign

Bingery just went through a full design-system pass — one token system
("warm light on a cool dark stage"), a semantic type scale (Fraunces /
Inter / JetBrains Mono), a documented motion vocabulary with full
`prefers-reduced-motion` support, and nine redesigned surfaces, from the
anime-detail hero to a stats page that replaced the usual GitHub-style
heatmap with a film-strip "Your rhythm" panel. Every change is purely
presentational: no data fetching, hooks, or tests were touched, and the
whole suite passes unmodified.
